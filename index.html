How to Build Your Wireframe - An Example



Wire-framing During Wire framing, you sketch out the different screens a user would possibly see during the course of the time he/she is using your web application. This clarifies what you need to build and how it might look after you've built it.

It might not necessarily encompass every single detail of how your final application might look like, but it sure should indicate the features you're going to have.

An example of this for our project is shown on the right. It consists of 3 screens.

  1. The Home Login-Signup screen
  2. The Dashboard Screen
  3. Add Borrower Screen
